Combining Existing Lists to Create a New List
You may create new lists from existing lists. To create a custom list from existing lists, follow these steps:
1.
At the top of the Manage List page, click on PatientDID List.
2.
Click the box next to the list name(s) to be combined to create a new list.
You cannot select more than two lists to use the Difference option.
3.
Enter a unique name for the new list you are creating, and then click the appropriate button:
•
Join selected combines two or more lists into one new list.
•
Intersect selected creates a new list from only the items that appear on more than one selected list.
•
Difference creates up to two lists each comprising items that appeared in one of the selected lists. For example, if you select all_males and all_stage_IIA, the new lists are “all_males-all_stage_IIA” comprising the items that appeared in the all_males list only and “all_stage_IIA-all_males” comprising the items appearing in the all_stage_IIA list only.
The new list appears on the Manage Lists page and in the blue panel in red .
