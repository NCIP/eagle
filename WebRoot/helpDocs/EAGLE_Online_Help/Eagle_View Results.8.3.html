Class Comparison Report
The Class Comparison report page lists the query specifications for this analysis, including the selected comparision groups. The table lists the group average, fold change, and p-value based on the parameters that you selected. The output varies based on the statistical method chosen.
For T-test Statistical Method analysis, the Class Comparison report is as follows.
•
The report displays the group average, where the numerator is the mean of log(base 2) expression signals from the samples in the first group. The denominator is the mean of log(base 2) expression signals from the samples in the second group.
•
The fold change for the reporter between the selected groups appears along with p-value.
•
Gene symbol annotations appear for each reporter. To obtain extensive annotations, click the in the upper right-hand corner of the report.
For F-test Statistical Method analysis, the difference is that in the results there is a group average displayed for each group. For more information, see F-test: One Way ANOVA.
For a GLM Generalized Linear Model analysis, the report identifies the genes showing statistically significant differences across the Baseline group and multiple Selected Groups. For more information, see GLM or Generalized Linear Model Details.
