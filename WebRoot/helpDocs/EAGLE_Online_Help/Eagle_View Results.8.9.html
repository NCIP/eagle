Downloading a Report
To download a report to a spreadsheet file, follow these steps:
1.
Click the button in the upper right-hand corner.
2.
Specify whether to save the file to a spreasheet or open the file in Notepad.
3.
To save the file, enter a name for the spreadsheet, and the file is saved to your computer.
