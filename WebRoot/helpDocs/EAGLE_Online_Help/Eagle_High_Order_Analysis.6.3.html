Performing a Class Comparison
To create a High Order Analysis with Class Comparisons, follow these steps:
1.
On the Class Comparison page, in the Analysis Name box, you are required to enter a name for the analysis.
If you enter the same name as a previous query during the current session, _copy# is appended to the report name in View Results.

Selected Groups
To select the group to use in the class comparison, follow these steps:
â€¢
Select a group in the Existing Groups box.
â€¢
Double-click the Existing Group, or click the >> button to move the group to the Selected Group list. Click the << button to remove it.
Select Constraint
Fold Change
Note: If you select the GLM Statistical Method, Fold Change is not available.
Specify the threshold for the differential regulation. This returns differential expression ratios between tumor and non-tumor samples for a particular reporter.
Less than or equal to 2.0 is the default.
 
p-Value
Note: If you select the GLM Statistical Method, p-Value is not available.
Specify the probability for obtaining the differences in expression values between tumor (or a subtype of tumor) and non-tumor samples.
Less than or equal to 0.05 is the default.
