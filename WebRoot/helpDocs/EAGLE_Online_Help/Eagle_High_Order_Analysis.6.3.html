Performing a Class Comparison
To create a High Order Analysis with Class Comparisons, follow these steps:
1.
On the Class Comparison page, in the Analysis Name box, you are required to enter a name for the analysis.
If you enter the same name as a previous query during the current session, _copy# is appended to the report name in View Results.
2.
The following table lists the available criteria:

Statistical Method
 
Drop down the list box, and select the appropriate statistical method:
•
T-test: Two Sample Test identifies genes showing statistically significant differences between two samples (Baseline and a Selected Group).
•
F-test: One Way ANOVA identifies genes showing statistically significant differences across two or more groups with no Baseline group. For more information, see F-test: One Way ANOVA.
•
Generalized Linear model with/without covariate adjustment GLM identifies genes showing statistically significant differences across the Baseline group and multiple Selected Groups. For more information, see GLM or Generalized Linear Model Details.

Baseline Group
Note: If you select the F-test Statistical Method, the Baseline Group is unavailable.
To select a baseline group, follow these steps:
•
Select a group in the Existing Groups box.
•
Click the Baseline>> button to move the group to the Baseline Group list. Click the <<Baseline button to remove it.
 
Selected Groups
To select the group to use in the class comparison, follow these steps:
•
Select a group in the Existing Groups box.
•
Double-click the Existing Group, or click the >> button to move the group to the Selected Group list. Click the << button to remove it.
Select Constraint
Fold Change
Note: If you select the GLM Statistical Method, Fold Change is not available.
Specify the threshold for the differential regulation. This returns differential expression ratios between tumor and non-tumor samples for a particular reporter.
Less than or equal to 2.0 is the default.
 
p-Value
Note: If you select the GLM Statistical Method, p-Value is not available.
Specify the probability for obtaining the differences in expression values between tumor (or a subtype of tumor) and non-tumor samples.
Less than or equal to 0.05 is the default.
3.
To submit your criteria and generate a Class Comparison report, click the Submit button.
You can also use the other buttons as follows:
•
To clear the values entered on the page and enter new values, click the Clear button.
•
To clear the values and return to the Analysis page, click the Cancel button.
The report results appear in a new window. You can access the report from the View Results page (see Class Comparison Report).
