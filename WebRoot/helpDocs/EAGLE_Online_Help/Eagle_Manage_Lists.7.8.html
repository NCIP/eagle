Exporting a List
To export the list of patient identifiers to a spreadsheet, follow these steps:
1.
At the top of the Manage List page, click PatientDID List.
2.
Find a list to be exported, and click the details icon to display all of the items in the list.
3.
Click the export list link.
4.
Open the file in Notepad or save the patient identifiers to a spreadsheet file.
To export a list from the side bar, double-click the list name. Open the file in Notepad or save the patient identifiers to a spreadsheet file.
