Performing an Epidemiology (EPI) Query
To define an Epidemiology (EPI) query, follow these steps:
1.
On the Epidemiology Query page, in the Query Name box, you are required to enter a name for the query.
If you enter the same name as a previous query during the current session, _copy# is appended to the report name in View Results.
2.
The following table lists the available search criteria.
At the top of the page, you can click a link, such as “Behaviorial,” to jump to that section of the EPI page.

3.
To submit your query and generate a EPI Query report, click the Submit Query button.
You can also use the other buttons as follows:
•
To clear the values entered on the page and enter new values, click the Clear button.
•
To clear the values and return to the Advanced Search page, click the Cancel button.
The report results appear in a new window. You can also access the report from the View Results page (see Epidemiology (EPI) Query Report).
