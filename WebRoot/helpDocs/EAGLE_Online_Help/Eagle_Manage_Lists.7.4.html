Combining Existing Lists to Create a New List
You may create new lists from existing lists. To create a custom list from existing lists, follow these steps:
1.
At the top of the Manage List page, click on PatientDID List.
2.
Click the box next to the list name(s) to be combined to create a new list.
You cannot select more than two lists to use the Difference option.
3.
At the bottom of the list names, enter a unique name for the new list(s) you are creating, and then click the appropriate button:
•
Join selected combines two or more lists into one new list.
•
Intersect selected creates a new list from only the items that appear on more than one selected list.
•
Difference creates up to two lists each comprising items that appeared in one of the selected lists. The first list comprising the items that appeared in the first list only and the second list comprising the items appearing in the second list only.
The new list name appears on the Manage Lists page and in the side bar in red . If you hover over the name in the side bar, the list’s data items appear.
When you hover over the new list name on the Manage List page, the creation date is updated, but the new list will not keep the previous list’s comments.
