EAGLE’s Opening Page
EAGLE’s opening page enables you to perform the following tasks:
•
Log in and out of EAGLE.
•
Display complete EAGLE online help with the help menu option.
•
Download an online version of the EAGLE User’s Guide with the user guide menu option.
•
Obtain EAGLE applicaton help with the support menu option.
For more information about the EAGLE workspace, see EAGLE Workspace.
