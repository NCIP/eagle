Saving Samples to a PatientDID List (EPI Report)
From a EPI report, you can save samples to a PatientDID list. Once saved, the PatientDID list will appear in red in the blue side bar. Use the Manage Lists function to further modify the list.
1.
There are two ways to select samples on the report:
•
To select an individual sample, select the box in the Patient DID column.
•
To select all of the samples, select the Check All box.
To clear all of the samples, deselect the Check all box.
2.
To save the selected samples, enter a unique name for the patient list next to Save Patient List.
3.
Clickthe Save Patient List button. Sample List Saved appears.
4.
Click the OK button.
Once saved, the sample set name is listed in red in the side bar.
