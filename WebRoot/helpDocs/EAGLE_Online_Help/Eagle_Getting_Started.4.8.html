EAGLE Main Menu and Tabs
When you log into EAGLE, the Main Menu appears on the Home page. You can access all EAGLE functions from the Main Menu or use the EAGLE tabs shown below.
The following table describes each EAGLE function.
Tab Name
Function
Advanced Search
Create an Epidemiology (EPI) query. For more information, see Search Overview.
Analysis
Run high order analyses, including a class comparison. For more information, see Analysis Overview.
Manage Lists
Manage user- or study-defined patient identifier lists. For more information, see Managing Lists Overview.
View Results
View reports from the Advanced Search an Analysis functions. For more information, see View Results Overview.
