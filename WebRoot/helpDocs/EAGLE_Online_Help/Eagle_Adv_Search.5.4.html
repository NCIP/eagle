Performing an SNP Query
The Single nucleotide polymorphisms (SNP) query enables you to generate a count of each type of SNP call for a designated group, click the Submit Query button.
1.
On the SNP Query page, in the Query Name box, you are required to enter a name for the query.
If you enter the same name as a previous query during the current session, _copy# is appended to the report name in View Results.
2.
The following table lists the available search criteria.
Criteria
Item Name
Special Instructions
Patients
(list box)
Drop down the list box and select a patient group to query. These groups are the PatientDID Lists shown on the EAGLE Side Bar.
SNP
(list box)
Type the dbSNP id or the common name for the SNP. The list box includes an automatic complete feature that enables you to review the identifiers fulfilling the typed values. Use the arrow keys to find an identifier.
3.
To submit your query and generate a SNP Query report, click the Submit Query button.
You can also use the other buttons as follows:
•
To clear the values entered on the page and enter new values, click the Clear button.
•
To clear the values and return to the Advanced Search page, click the Cancel button.
The report results appear in a new window. You can also access the report from the View Results page (see SNP Query Report).
