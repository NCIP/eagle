Advanced Search Overview
The Advanced Search page enables you to define queries based on your specifications. Under Select an Advanced Search Type heading, click a link to start defining a query:
•
Epidemiology (EPI) Query (see Performing an Epidemiology (EPI) Query)
•
Single nucleotide polymorphisms (SNP) Query (see Performing an SNP Query)
An advanced search generates results that you can review on the View Results page (see View Results Overview).
